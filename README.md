 # 🍕 Pizza Sales Data Analysis using PostgreSQL & Power BI

## 📌 Project Overview
This project demonstrates an end-to-end data analytics workflow using PostgreSQL and Power BI.
The project begins with storing pizza sales data in PostgreSQL, performing SQL analysis to calculate KPIs and business metrics, and finally visualizing the results in an interactive Power BI dashboard.

## Key Business Insights
- Large pizzas generated the highest sales contribution.
- Classic pizzas accounted for a significant share of total sales.
- The Pepperoni Pizza ranked among the top-selling products.
- The Brie Carre Pizza was the lowest-performing pizza.
- Revenue fluctuated across months, highlighting seasonal demand patterns.
